# spm-deps - Dependency Tree Repository

Machine-readable dependency data for Solaris 7 SPARC packages.

This repo is auto-generated by `spm export-deps` and provides a
centralized dependency graph for TGCware and custom SPARC packages.

## Structure

```
deps/
├── tgcware/           # TGCware package dependencies
│   ├── curl.json
│   ├── bash.json
│   └── ...
├── firefly128/        # GitHub release package dependencies
│   ├── pizzafool.json
│   └── sparccord.json
└── metadata.json      # Index metadata
```

## Format

Each package file contains:

```json
{
  "name": "curl",
  "pkg_code": "TGCcurl",
  "version": "8.18.0",
  "release": 1,
  "arch": "sparc",
  "source_type": "tgcware",
  "source_url": "https://github.com/AstroVPK/tgcwarev2-for-solaris/tree/master/curl",
  "deps": [
    { "code": "TGCgtext", "name": "gettext" },
    { "code": "TGClgcc1", "name": "libgcc" },
    { "code": "TGClicnv", "name": "libiconv" },
    { "code": "TGClidn2", "name": "libidn2" },
    { "code": "TGClssh2", "name": "libssh2" },
    { "code": "TGCngh2", "name": "nghttp2" },
    { "code": "TGCssl3", "name": "openssl" },
    { "code": "TGCzlib", "name": "zlib" }
  ]
}
```

## Usage

This data can be consumed by CI/CD, package build systems, or
web-based dependency viewers.
